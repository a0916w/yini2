# Handoff: YiniTV 短剧App「果橙·活力」重设计

## Overview
YiniTV 是一个竖屏短剧播放 App（类似抖音/红果短剧）。本次交付为全新视觉方向「果橙·活力」：浅橙清新底色、渐变头部、胶囊分类、榜单前三领奖台，共 5 个 Tab + 沉浸式播放。目标是替换现有 App 的样式（结构/功能基本保持）。

## About the Design Files
本包中的文件是 **HTML 设计参考稿**（可交互原型），不是可直接上线的生产代码。任务是：**在目标代码库现有技术栈中（React / Vue / Flutter / 原生等）按其既有模式重新实现这些设计**。如果项目还没有前端环境，请选择最合适的框架实现。`yinitv-app.dc.html` 是设计源文件；`ios-frame.jsx` 只是预览用的 iPhone 机框，**不需要实现**。

## Fidelity
**High-fidelity（高保真）**：颜色、字号、间距、圆角、阴影均为最终值，按下方 token 与逐屏描述像素级还原。剧集封面目前为「色块 + 文字」占位，上线时替换为真实竖版剧照（比例约 3:4 至 9:16），保留封面上的角标/评分等覆盖元素。

## Screens / Views（5 个 Tab + 播放页）

### 1. 首页（feed，默认 Tab，最左）
抖音式全屏竖滑播放器，深底 `#0B0B0D`。
- 结构：全屏容器 overflow hidden，内部纵向堆叠 N 个 100% 高度的视频页，`transform: translateY(-index*100%)`，过渡 `0.5s cubic-bezier(0.22,1,0.36,1)`
- 顶部（状态栏下方）：居中「推荐」16px/700 白；右侧「弹幕 开」chip（黑30%底、blur、全圆）
- 右侧操作栏（右14px，距底200px，纵向 gap 20px）：创作者头像 46px 圆（白2px描边 + 底部主色 "+" 关注角标 18px）、点赞❤+数字、评论💬+数字、收藏⭐、分享➦，图标白色 28-30px，数字 11px/600
- 左下信息区（左16px，右让出80px，距底132px）：@剧场名 15px/800 白、剧名 15px、「选集 · 全N集」chip（白16%底+blur，radius 8）、♪ 原声一行（白70%，超出省略）
- 底部进度条（左右16px，距底96px）：3px 白25%轨道 + 白色已播 + 10px 圆点手柄；下方 3:09 / 9:47 时间 10px 白60%
- 「上滑看下一部」提示 10px 白45%，位于进度条上方（距底132px）
- 弹幕开时：顶部区域漂浮 1-2 条弹幕 chip（黑25%底+blur 全圆）

### 2. 剧场（home）
浅橙渐变从页顶淡出。
- 顶栏：logo YiniTV + 「签到有礼」白底主色 chip + 搜索圆钮 32px
- 分类胶囊横滑：全部（选中=主色底白字）/ 逆袭 / 甜宠 / 悬疑 / 古风（白85%底、#7A6B5C 字）
- 今日主打 Hero：全宽 200px 高 24px 圆角封面卡；左上「今日主打」白底主色 chip；左下剧名 24px/800 白、★9.0（`#FFD27A`）· 播放量 · 集数 12px、主色「立即看」按钮（▶ 图标 + 全圆 + 主色阴影）
- 新剧首发：标题行（19px/800 + 「每日上新」小chip + 右侧"全部 ›"）；横滑封面 112×150px r18，左上「新」主色角标，封面内底部剧名（12px/700 白，最多3行）+ ▶ 播放数
- 大家都在看：2 列网格，170px 高 r20 封面卡，右上「★ 9.6」主色 chip，底部剧名 + 类型·在看数

### 3. 榜单（rank）
顶部浅橙渐变。标题「热播总榜」24px/800 + 副标题「每小时更新 · 全网实时播放热度」。
- 榜单切换胶囊：飙升榜（主色底白字）/ 热播榜 / 新剧榜
- 前三领奖台：三列并排底对齐，第1名居中最高（140px 封面 + 顶部皇冠 SVG `#FFB020` + 阴影），2/3 名 118/106px；每列下方 名次数字（italic 900）+ 热度值（主色 11px/700）
- 4-5名列表：`#FFF6EE` 暖底卡 r16，行内：名次数字、44×58 封面缩略、剧名 14px/700 + "类型 · 热度 N"、右侧主色「看剧」胶囊按钮

### 4. 专题（topic）
标题「全部专题」+ 副标题「编辑精选片单 · 每周四更新」。
- 全宽渐变专题卡（r22，各自色系渐变+同色阴影）：左侧专题名 20px/800 白、"N 部剧 · N 热度" 白75%、白底「去看看 ›」胶囊（文字用卡片深色）；右下角斜放的迷你封面（白14%底、rotate 12deg，多部剧时叠两张）

### 5. 我的（me）
- 顶部主色渐变头部（`150deg #FF8A3C→#FF6000`，底部圆角 30px）：「个人中心」22px/800 白、头像 56px 白底圆（主色字母）、未登录/点击登录注册、白22%「登录」chip

- 底部「登录」主色渐变全圆按钮

### 6. 播放页（从任意剧集卡进入）
与首页 feed 单页 UI 完全一致，另加左上角返回圆钮（34px，黑30%底+blur，白色 ‹ 箭头）。以全屏覆盖层呈现，返回回到来源页。

## 底部 Tab 栏
5 项：首页 / 剧场 / 榜单 / 专题 / 我的。图标 22px 线性（1.8 stroke），标签 10px/700。
- 浅色态（剧场/榜单/专题/我的）：`rgba(255,255,255,0.94)` + blur16 + 顶部 1px `#F6EDE4`；激活 `#FF6000`，未激活 `#B8AFA6`
- 深色态（首页 feed）：`rgba(12,12,14,0.88)` + 顶部 1px 白8%；未激活 `rgba(255,255,255,0.55)`
- 切换 Tab 时底栏颜色随之切换

## Interactions & Behavior
- Tab 切换：点击底栏切换 5 个页面，激活色即时变化
- 首页 feed：上滑/滚轮 → 下一部，下滑 → 上一部；450-550ms 节流防连滑；translateY 弹性过渡；到达首/尾不再滚动
- 进入播放：点击任意剧集卡片、「立即看」「看剧」「去看看」→ 全屏播放页；左上返回关闭
- 所有可点击卡片 cursor:pointer；按钮无 loading 态要求
- 弹幕开关：全局布尔，关闭时隐藏漂浮弹幕

## State Management
- `currentTab: 'feed' | 'home' | 'rank' | 'topic' | 'me'`（默认 'feed'）
- `feedIndex: number`（首页当前视频序号，带节流锁）
- `playingDrama: Drama | null`（播放页覆盖层；Drama: {title, coverColor/coverImage, studio, likes, comments, episodes, tag}）
- 数据获取：首页 feed 推荐流、剧场首页栏目、榜单（3 个榜）、专题列表——均为服务端数据，原型中为写死示例
- 建议持久化：播放进度、观看记录
